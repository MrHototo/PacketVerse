Filter syntax

  • dns — bare protocol / tag match
  • ip.addr==10.0.0.5 — match either src or dst
  • ip.src==10.0.0.5 / ip.dst==8.8.8.8
  • tcp.port==443, udp.port==53
  • bytes>10000, packets>50
  • ip.addr ~ /^10\./ — regex match
  • dns and udp.port==53, (tls or http) and bytes>1000
  • not arp, !broadcast

Plain text with no operators falls back to a simple substring search.
